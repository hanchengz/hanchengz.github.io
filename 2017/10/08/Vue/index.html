Vue实例

每个Vue应用都是通过Vue函数创建一个新的Vue实例开始的。

数据与方法

在创建一个新的Vue实例时，可以传入一个选项对象，来创建我们想要的行为。在创建时它向Vue的响应式系统中加入了data对象中的所有属性，当其中的属性的值发生变化时，视图也会实时的响应，更新为新的状态。
值得注意的是，当数据发生改变时，视图会重新渲染，但是仅有实例被创建时data中存在的属性是响应式的。这意味着如果我们在实例化一个Vue对象后再添加一个新的属性，那么这个新增的属性发生改变时不会触发视图的更新。
除了data属性，Vue实例还暴露了一些有用实例属性与方法。它们都有前缀$，以便与用户定义的属性区分开来。

实例生命周期

每个Vue实例在创建之前都要经过一系列的初始化过程。例如需要设置数据监听，编译模版，挂载实例到dom，在数据变化时更新dom等。同时也会运行一些叫做生命周期钩子的函数。
例如created钩子可以用来在一个实例被创建后执行代码。

还有一些其他的钩子，在实例生命周期不同的场景下调用。如mounted、updated、destroyed。钩子的this指向调用它的Vue实例

不要在选项属性或回调上使用箭头函数，比如created: () => console.log(this.msg),或vm.$watch(‘a’,newValue => this.myMethod())。因为箭头函数是和父级上下文绑定在一起的，this不会是预想中的值。

模版语法

插值

数据绑定最常见的形式就是使用“Mustache”语法（双大括号）的文本插值。

Mustache标签将会被替换为相应数据对象上msg属性的值。当msg的值发生改变时，插值处的的内容都会更新。
也可以通过使用v-once指令来执行一次性的插值，这样一来数据发生改变时，插值处的内容也不会更新，但这会影响到这个节点的所有数据绑定。

原始HTML

双大括号会将数据解释为普通文本，而非HTML代码，如果需要输出原始的HTML，则要使用v-html指令。

这样一来这个div的内容将会被替换为属性值rawHtml，要注意的是直接作为HTMl，会忽略解析属性值中的数据绑定。同时不能使用v-html来复合局部模版。

特性

Mustache语法不能作用在HTML特性上，需要使用v-bind指令来完成。

使用javascript表达式

对于所有的数据绑定，Vue.js都提供了完全的javascript表达式支持。

模版表达式都放在沙盒中，只能访问全局变量的一个白名单，比如Math和Date。不能在模版表达式中试图访问用户定义的全局变量。

指令

指令是带有v-前缀的特殊属性。指令属性的值预期是单个JavaScript表达式（不包括v-for）。指令的作用是当表达式的值发生改变时，将其产生的连带影响，响应式地作用于DOM。

参数

一些指令可以接收一个参数，在指令名称之后以冒号表示。

在这里href是参数，使用v-bind指令将该元素的href属性与表达式url的值绑定在一起。
v-on用于监听DOM事件，参数为事件名。

修饰符

用于指出一个指令应该以特殊方式绑定。如 .prevent修饰符告诉v-on指令对于触发的事件调用event.preventDefault()
